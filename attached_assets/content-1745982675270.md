# EVE Frontier API

[![Pub](https://img.shields.io/pub/v/evefrontier_api.svg)](https://pub.dartlang.org/packages/evefrontier_api)

Library for working with [EVE Frontier](https://evefrontier.com/) online game api.

[![BuyMeACoffee](https://img.buymeacoffee.com/button-api/?text=Donate&emoji=&slug=dip.dev&button_colour=29b6f6&font_colour=000000&font_family=Cookie&outline_colour=000000&coffee_colour=FFDD00)](https://buymeacoffee.com/dip.dev)

## Getting Started

To use this package, add `evefrontier_api` as a [dependency in your pubspec.yaml file](https://flutter.io/platform-plugins/).

```yaml hljs
dependencies:
  evefrontier_api: <latest_version>

```

## Using

```dart hljs
import 'package:evefrontier_api/evefrontier_api.dart';

```

## Example

```dart hljs
  final api = EVEFrontierAPI();
  api.getSolarSystems().then((value) {
    print(value.length);
  });

```

## Generate models

```bash hljs
dart run build_runner build -d

```

[Swagger API Documentation](https://docs.evefrontier.com/SwaggerWorldApi).
